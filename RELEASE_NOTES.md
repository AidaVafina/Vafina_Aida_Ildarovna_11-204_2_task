# Release Notes

## v1.0.0 — 2026-03-04

- Добавлен скрипт `src/process_documents.py` для токенизации и лемматизации.
- Реализована очистка мусорных токенов:
  - удаление чисел;
  - удаление смешанных токенов (буквы+цифры);
  - удаление служебных частей речи (союзы, предлоги, частицы, междометия).
- Добавлена обработка `.txt`, `.html`, `.htm`.
- Реализована генерация отдельных файлов токенов и лемм для каждого документа.
- Добавлены примеры входных документов в `data/raw/`.
- Добавлены готовые выходные файлы в `data/results/`.
- Добавлена документация: `README.md`, `DEPLOYMENT_MANUAL.md`, `RELEASE_NOTES.md`.
