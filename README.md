# Vafina_Aida_Ildarovna_11-204_2_task

Решение задания по предмету «Информационный поиск»:

1. Токенизация сохраненных документов.
2. Удаление дублей, чисел, мусорных токенов, союзов и предлогов.
3. Лемматизация и группировка токенов по леммам.
4. Создание отдельных файлов токенов и лемм для каждого входного документа.

## Стек

- Python 3.11+
- `pymorphy3` (лемматизация)
- `beautifulsoup4` (очистка HTML)

## Структура проекта

- `src/process_documents.py` — основной скрипт.
- `data/raw/` — входные документы (`.txt`, `.html`, `.htm`).
- `data/results/tokens/` — выходные файлы токенов по каждому документу.
- `data/results/lemmas/` — выходные файлы лемм по каждому документу.
- `DEPLOYMENT_MANUAL.md` — инструкция по запуску.
- `RELEASE_NOTES.md` — история релизов.

## Формат результатов

### 1) Файлы токенов

Для каждого документа создается файл вида:

`<имя_документа>_tokens.txt`

Формат строки:

`<токен>\n`

### 2) Файлы лемм

Для каждого документа создается файл вида:

`<имя_документа>_lemmas.txt`

Формат строки:

`<лемма><пробел><токен1><пробел><токен2>...<пробел><токенN>\n`

## Быстрый запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/process_documents.py
```

После выполнения результаты будут в:

- `data/results/tokens/`
- `data/results/lemmas/`

## Важно для проверки

- Главный код находится в ветке `master`.
- На каждый входной документ создаются отдельные файлы токенов и лемм.
- Используется именно лемматизация (`pymorphy3`), а не стемминг.
